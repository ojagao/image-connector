# 画像結合ツール

2枚の画像を横に並べて1枚の画像に結合するWebツールです。
ブラウザだけで動作するので、インストール不要ですぐに使えます。

## 使い方

### 1. ツールを開く

`index.html` をブラウザで開いてください。

- **Windows**: ファイルをダブルクリック、または右クリック → 「プログラムから開く」 → お好きなブラウザを選択
- **Mac**: ファイルをダブルクリック、または右クリック → 「このアプリケーションで開く」 → お好きなブラウザを選択

推奨ブラウザ: Chrome、Edge、Safari、Firefox

### 2. 画像を選択する

#### 方法1: 2枚同時に選択
1. 「結合する画像を選ぶ」ボタンをクリック
2. ファイル選択ダイアログで2枚の画像を選択（Ctrl/Cmd を押しながらクリック）
3. 「開く」をクリック
4. → 編集画面が開きます

#### 方法2: 1枚ずつ選択
1. 「結合する画像を選ぶ」ボタンをクリック
2. 1枚目の画像を選択して「開く」をクリック
3. → 1枚目のプレビューが表示されます
4. 「2枚目を選ぶ」ボタンをクリック
5. 2枚目の画像を選択して「開く」をクリック
6. → 編集画面が開きます

### 3. 画像を編集する

編集画面では、画像をドラッグして位置を調整できます。

- **移動**: 画像をドラッグ
- **サイズ変更**: 画像の角をドラッグ
- **回転**: 画像の上部の回転ハンドルをドラッグ

### 4. ダウンロードする

1. 画像の配置が決まったら「ダウンロード」ボタンをクリック
2. `connected-image.png` という名前でダウンロードされます

> **ポイント**: ダウンロードされる画像は高解像度（4倍）で、画像が配置されている部分だけが自動的にトリミングされます。

### 5. やり直す

最初からやり直したい場合は「最初に戻る」ボタンをクリックしてください。

## 対応ファイル形式

- **画像ファイル**: JPG, PNG, GIF, WebP など
- **PDFファイル**: PDF の1ページ目を画像として読み込みます

---

## 機能

### ✨ 主な機能

- 画像の自由な配置・サイズ変更・回転
- 画像が存在しない余白部分の自動トリミング
- 高解像度（4倍）での書き出し
- PDFファイル対応（1ページ目を自動変換）

### 🎯 初期配置

画像は自動的に以下のように配置されます：

- 各画像は Canvas の 50% の幅に調整されます
- 縦長の画像の場合は Canvas の高さに合わせて縮小されます
- 2枚の画像は中央で接するように配置されます

## よくある質問

**Q. インターネット接続は必要ですか？**
A. 初回読み込み時のみ必要です（Fabric.js、PDF.jsをCDNから読み込むため）。一度読み込めばオフラインでも動作します。

**Q. 画像データはどこかに送信されますか？**
A. いいえ。すべての処理はブラウザ内で完結します。画像データが外部に送信されることはありません。

**Q. 3枚以上の画像を結合できますか？**
A. 現在のバージョンでは2枚までです。

**Q. 縦に並べることはできますか？**
A. 編集画面で画像を自由に配置できるので、縦並びも可能です。

**Q. ダウンロードした画像の画質が悪い場合は？**
A. ダウンロード時に4倍の解像度で書き出しているため、通常は高品質です。元の画像が低解像度の場合は、それ以上の品質にはなりません。

## トラブルシューティング

### 画像が表示されない
- ファイル形式が対応しているか確認してください
- ブラウザのコンソール（F12キー）でエラーが出ていないか確認してください

### PDFが読み込めない
- PDF.jsライブラリの読み込みに失敗している可能性があります
- インターネット接続を確認してください
- ページを再読み込み（F5キー）してください

### ダウンロードボタンが反応しない
- 画像が正しく配置されているか確認してください
- ブラウザのポップアップブロック設定を確認してください
